    RSVP App

    Treehouse's "DOM Scripting By Example" Course

    Helps track party RSVPs. In the RSVP app you can enter names in this text field, click Submit, and the names appear on the invited list. Once we hear from invitees, we can check them off, and the confirmed text and outline change color to show that it's been selected. We can then filter out the invitees who haven't responded yet by checking this checkbox. And if we enter someone's name incorrectly, we can fix it by clicking the edit button, correcting the name and saving it. Finally, if we need to remove a name, we can simply click remove.
