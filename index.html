About Me

I'm a Las Vegas-based front-end designer who works to design and code beautiful websites.

Before starting a career in tech, I was a sci-fi obsessed academic with an MA in English from the University of Rochester.

After paying attention to my obsession with design, I enrolled in Skillcrush's Web Designer blueprint which led to visual design, responsive web development, enrolling in courses with Treehouse and Udemy.

I call myself a Front-End Designer because I design and code the websites I create to ensure that I can develop the design as the code is being written and vice versa.
